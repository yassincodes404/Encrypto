About This Project

Project Overview

Encrypto is a secure web application focused on providing robust cryptographic tools. It aims to facilitate secure communication and data protection through user-friendly interfaces backed by industry-standard encryption algorithms and security protocols, ensuring data privacy and integrity.

Key Features
  • Advanced Encryption Tools

    Interactive tools for SHA-1 hashing, AES-CTR symmetric encryption, and RSA asymmetric encryption.

  • Secure Real-time Chat

    Encrypted messaging using AES-CTR with digital signatures (RSA-SHA1) for authenticity and integrity.

  • User Account Management

    Secure registration and login with password hashing. Users have profiles with associated RSA key pairs.

  • Multi-language Interface

    Support for English and Arabic, adapting layout direction (LTR/RTL) accordingly.

  • Customizable Theme

    User-selectable light and dark themes for personalized visual experience.

Security Implementation Details

Security is a core aspect of Encrypto. Key measures include:

  • Password Hashing: Using Werkzeug's `generate_password_hash` for secure password storage.
  • RSA Key Pairs: Each user gets a 2048-bit RSA key pair upon registration for encryption and digital signatures.
  • Digital Signatures: Messages are signed with the sender's private RSA key and SHA-1 hash for authenticity verification.
  • Symmetric Encryption (Chat): AES-256 in CTR mode used for message content encryption in chat (Note: Hybrid encryption approach is ideal for general data).
  • Session Management: Flask-Login handles secure user sessions.
  • CSRF Protection (Implied): Flask-WTF or similar mechanisms are recommended for forms (not explicitly shown here).

Note: While SHA-1 is used for demonstration in signatures, stronger hash functions like SHA-256 are recommended for new production systems.
